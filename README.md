# 🍽️ Zomato Analytics Project

## 📊 Project Overview
This project analyzes restaurant data from Zomato to uncover insights about restaurant performance, popular cuisines, pricing trends, and customer preferences. The analysis was performed using SQL, Excel, Power BI, and Tableau to transform raw data into meaningful business insights.

---

## 🎯 Business Problem
Food delivery platforms need insights into restaurant performance and customer preferences.  
This project helps answer questions such as:
- Which cuisines are most popular?
- Which restaurants have the highest ratings?
- How does price range affect customer ratings?
- How many restaurants offer online delivery?

---

## 🛠 Tools & Technologies
- SQL
- Excel
- Power BI
- Tableau

---

## 📂 Dataset
The dataset contains restaurant information such as:
- Restaurant Name
- Location
- Cuisine Type
- Rating
- Price Range
- Online Delivery
- Table Booking


## 📈 Key Insights
- Most restaurants belong to a few popular cuisine categories.
- Higher price ranges tend to have slightly better ratings.
- Online delivery is offered by a large percentage of restaurants.
- Certain locations have a much higher concentration of restaurants.

---

## 🚀 Project Outcome
This project demonstrates how data analytics tools can be used to analyze restaurant data and generate business insights that help companies improve decision-making.
